# Guia de Suporte de TI

Guia básico de solução de problemas para questões comuns de TI.

## Problemas do Windows
- Verificar a conexão de rede
- Reiniciar o computador
- Atualizar os drivers
- Verificar o Gerenciador de Tarefas

## Solução de Problemas de Rede
- Verificar a conexão por cabo ou Wi-Fi
- Executar ipconfig /all
- Reiniciar o roteador

## Problemas de Software
- Reinstalar o aplicativo
- Verificar a compatibilidade
- Limpar o cache
